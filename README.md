## Events hinzufügen oder ändern

Die `EVENTS`-Liste in `index.html` bearbeiten:

```js
{
  date: "2026-06-15",          // Format: YYYY-MM-DD
  title: "Name des Events",
  cat: "demo",                 // demo | antifa | klima | lgbtq | rad | workshop
  url: "https://...",          // Link zur Originalseite (Klick im Kalender)
  info: "14:00 Uhr · Ort"     // Kurzbeschreibung (Tooltip + Eventliste)
}
```
